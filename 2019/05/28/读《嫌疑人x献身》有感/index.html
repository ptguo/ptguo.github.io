读《嫌疑人x献身》有感

发表于 2019-05-28 更新于 2019-09-18 分类于 文学

这是我读的第二本东野圭吾的作品，第一本是《白夜行》。本以为两本会是很相似的风格，但却不然。刚读《白夜行》的时候或许希望参考社会背景，会很想知道那是日本哪个时代发生的事情。而《嫌疑人x的献身》就没有这种冲动，因为这里所讨论的问题在大多数时代都会存在和发生，具有通用性。从写作角度来看二者截然不同。

前面的铺叙占用了四分之三的内容，作者必须想办法让读者度过这漫长的铺叙才能体会到后面瞬间解开谜团的快感，这一次他利用读者既想维护女主花冈靖子又希望草雉刑警能够根据线索解开谜底的矛盾心情吸引不断往前阅读。实际上，刑警和罪犯的较量也是全书的主要矛盾。但是直到最后的高潮部分，才恍然发现所谓“献身”的含义，这种点题方式不愧是东野圭吾。当罪犯不止一人时，会给调查带来极大的困难，利用不同的人在时间上的错位就可以简单地令不在场证明变得没有意义。这时老友汤川学认为，转变思路从犯罪动机的角度分析会更有利于调查。这类似于科学研究中的“还原论”，调查的目的是还原现场所发生的事情。当汤川学告诉女主真相的一瞬间，全书达到高潮部分，为何那么多人爱我我却依然不能幸福，从前甚至无法理解世间会有如此深情，在那么多铺垫之后，此时的这些提问才会显得如此深刻。幸福并不仅仅是得到爱，更是对爱的付出，而献身无疑体现着一种大爱无言。东野圭吾塑造的男主石神哲哉不仅仅在数学研究中无法被世人认可，甚至自己的爱也无法得到理解，却又是聪明绝顶之人，这种人如果把思路花在作案上那真是应了汤川学的话，但这也是无法摆脱的社会矛盾。

全书在男女主最后的相遇中戛然而止，虽然没有明确写，但是美里应该足够聪敏已经知道男主为她妈妈如何付出的，因此才会在出租车上有那段责备母亲的对话，后面这一切都足以去想象。

我一开始以为嫌疑人x的献身是男主三月十号安排好了后面所有的事情，然后在那个河边自杀了，警方发现的尸体是真正男主的尸体，而后面的男主另有其人（就是花五万雇了的技师），是男主伪装成富樫慎二，后自杀，让警方以为其实是自杀案件实际上除了身份也确实是自杀案就可以了。这不是才更符合所谓的“献身”嘛，没想到实际上他真去杀了个人。让后面工藤邦明和女主在一起也没啥不好，再写写他们后来怎么又不幸福了以及美里的故事甚至让她道出真正的案情可以再写一本小说。

总有一种感觉，《白夜行》是东野圭吾希望挑战的一种风格。
